**ContaX is a desktop application for managing your Contacts and Schedule.** It is a powerful tool optimized for use via a Command Line Interface (CLI), while incorporating Graphical User Interface (GUI) elements to make it user-friendly.

* If you are interested in using ContaX, head over to the [_Quick Start_ section of the **User Guide**](UserGuide.html#quick-start).
* If you are interested about developing ContaX, the [**Developer Guide**](DeveloperGuide.html) is a good place to start.
